Mark all documents matching the query as deleted. Uses the same query syntax as Search. Deletion is immediate; space is reclaimed on next Compact.
Administration

🗜 Compact

Merge small segments via size-tiered compaction. Physically removes deleted documents from storage.

🗑 Garbage Collect

Delete orphaned segment files from S3 that are no longer referenced by the active manifest.

💾 Flush

Force-flush the in-memory write buffer to a new immutable segment on S3 immediately.
